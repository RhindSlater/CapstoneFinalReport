Abstract

We have created a website to make a free platform for selling trading game cards (starting with Yu-Gi-Oh!) with a business model of providing an optional verification service fee which charges 10% of the total price of a sale, as well as running some ads. Users may choose to become a premium member by subscribing to a monthly fee of $11.99 which allows them to have limitless verification purchases and ad-free access to the website. We also have one minute long ‘blind auctions’ where users can bid on auctions that start at a set time that we, 2B Trading, will decide. The winner of an auction is not known until it is finished so that users will only bid as much as they think the card is worth.

The technology we used was ASP.NET MVC with Entity Framework Code First Migrations. The database is hosted by SmarterASP and it uses MSSQL. We have also implemented the use of AJAX, JQuery, and JavaScript. We used Azure Devops as our software managament system as it was easy to setup and use with good collaboration. Github was our main source of source control; Using either gitkraken or github desktop. Facebook and discord were used to communicate between team members and our product owner.

We decided to use the above mentioned technologies because they were the ones we were mainly taught at our course and the most comfortable with, while learning other skills as we progressed through the project.
